Detailed Description

By default, no configuration values are set.

SD/MMC Configuration

The following SD/MMC configurations are supported:

InstancePeripheralI/O SetBus Width
0SDMMC011-bit, 4-bit
1SDMMC111-bit, 4-bit

The SD/MMC applet on SAM9X60 does not support voltage switching, so the voltage configuration property is ignored.
